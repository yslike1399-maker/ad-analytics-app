# ad-analytics-app

광고 플랫폼 데이터를 **엑셀 파일** 또는 **스크린샷 이미지**로 입력받아,
효율 지표를 자동 계산하고 시각화해주는 브라우저 기반 분석 도구.

광고 플랫폼 API 연동 없이도, 마케터가 손에 들고 있는 자료(다운로드한 엑셀,
관리자 화면 캡쳐) 그대로 분석할 수 있는 것이 핵심.

---

## 핵심 기능

### 1. 엑셀 업로드 분석
네이버, 카카오, 인스타그램 등 광고 관리자에서 다운로드한 엑셀 파일을 업로드.
앱이 플랫폼별 컬럼 구조를 자동 감지해 **통합 스키마**로 정규화한 뒤 분석한다.

### 2. 스크린샷 AI 분석 (Claude Vision)
광고 관리자 화면을 캡쳐한 이미지를 Claude API에 전달해
노출/클릭/비용/매출 등 숫자를 자동 추출.
엑셀 다운로드가 번거롭거나, 빠르게 확인하고 싶을 때 사용한다.

### 3. 자동 지표 계산
- **ROAS** — 광고비 대비 매출 (매출 ÷ 광고비 × 100)
- **CTR** — 클릭률 (클릭 ÷ 노출 × 100)
- **CPC** — 클릭당 비용 (광고비 ÷ 클릭)
- **CPM** — 1000회 노출당 비용
- **전환율, ROI** 등

### 4. 시각화 대시보드
- 광고비 vs 매출 비교 차트
- 일별/주별 트렌드 그래프
- 플랫폼별 · 캠페인별 효율 비교
- 상위/하위 캠페인 순위

## 기술 스택

빌드 도구 없이 **단일 HTML 파일**로 동작하는 SPA 구조.
LACEROOM 통합관리 앱과 동일한 형식.

| 영역 | 선택 | 비고 |
|------|------|------|
| 프레임워크 | React 18 (UMD) | CDN 직접 로드, 빌드 없음 |
| 스타일 | Tailwind CSS (CDN) | 흑백 미니멀 디자인 시스템 |
| JSX 변환 | Babel Standalone | 브라우저에서 실시간 JSX 컴파일 |
| 엑셀 파싱 | SheetJS (xlsx) | 클라이언트 사이드 엑셀 처리 |
| AI Vision | Anthropic API (fetch) | 사용자가 입력한 API 키로 직접 호출 |
| 시각화 | Plotly.js (예정) | Phase 4에서 추가 |
| 데이터 저장 | localStorage | 설정 · 히스토리 영구 보관 |
| 백업 | JSON 내보내기/가져오기 | 사이드바 하단 버튼 |
| 배포 | GitHub Pages | 정적 호스팅 |

> **API 키 보안**: Anthropic API 키는 사용자가 [설정] 탭에서 입력하며,
> 해당 브라우저의 localStorage에만 저장된다. 외부로 전송되지 않으며,
> 다른 사람이 사이트에 접속해도 본인 키만 사용된다.

## 정규화된 데이터 스키마 (초안)

플랫폼이 달라도 결국 아래 한 형태로 통합:

| 필드 | 타입 | 설명 |
|------|------|------|
| platform | str | naver / kakao / instagram |
| campaign | str | 캠페인명 |
| date | date | 집계일 |
| impressions | int | 노출수 |
| clicks | int | 클릭수 |
| cost | int | 광고비 (원) |
| conversions | int | 전환수 |
| revenue | int | 매출 (원) |

엑셀 파서, Vision 추출기 모두 이 스키마로 변환한 뒤
분석 엔진에 넘긴다.

## 실행 방법

### 로컬 테스트
브라우저로 `index.html`을 직접 열거나, 간단한 정적 서버로 실행:

```powershell
# 옵션 A: Python 임시 서버
python -m http.server 8000

# 옵션 B: VS Code Live Server 확장 사용
```

`http://localhost:8000`에서 접근.

### GitHub Pages 배포
`main` 브랜치를 Pages 소스로 설정하면 즉시 배포된다.

```
Settings → Pages → Source: Deploy from a branch → main / (root)
```

배포 URL: `https://<username>.github.io/ad-analytics-app/`

## 데이터 백업

사이드바 하단의 **[내보내기]** 버튼으로 모든 데이터(설정, 히스토리)를
JSON 파일로 저장할 수 있다. **[가져오기]** 로 다른 PC에서 복원 가능.

브라우저 캐시를 비우면 localStorage 데이터가 사라지므로,
중요한 분석 결과는 주기적으로 내보내기 권장.
